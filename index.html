상담 문의 내역

블로그 주소 등록

네이버 블로그 주소만 넣으면 대표이미지와 제목을 자동으로 불러옵니다.

블로그 제목/내용 직접 입력 + 대표이미지 첨부 방식입니다.

프로젝트 등록

사진을 최대 5장까지 넣고, 등록 버튼을 한 번만 누르면 Firebase Storage에 저장됩니다. 목록은 빠른 썸네일, 클릭 확대는 원본 화질로 표시됩니다.

윈도우: Ctrl 클릭 / 맥: Command(⌘) 클릭으로 여러 장 선택 가능합니다. 자동 압축 저장됩니다.

시공영상 관리

유튜브 쇼츠/영상 주소, 제목, 설명, 썸네일을 등록하면 홈페이지 영상 섹션에 표시됩니다. 첫 번째 영상이 왼쪽 메인 영상으로 표시됩니다.
